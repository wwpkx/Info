# 线性代数
## 向量
- 各种概念
	- 向量(Vector), 是**一组有序的数**，$\vec{u}$
	- 向量的模，$||\vec{u}||$
	- 向量
		- 零向量，模为0，没有方向
		- 单位向量，模为1，任意方向
		- n维向量
			- 二维向量，平面
			- 三维向量，立体
		- 向量组，好多个同维的向量放在一起
- 分类
	- 行向量
	- 列向量
- 基本运算
	- 加法
	- 乘法
		- **数乘**，标量乘法，**结果为向量**
		- **点乘**，向量内积，**结果为标量**
- 等价向量组
	- 比如：直角坐标系旋转45度
- 范数
